## 用自己的话回答

- 什么是特征，什么是标签：特征是模型用来做判断的输入信息（如使用时长、是否投诉等用户属性），标签是我们希望模型预测的答案（在本例中是"是否流失"）
- 为什么要保留测试集：测试集模拟真实场景中从未见过的新用户，用来检验模型是否真的学会了规律而不是死记硬背训练数据
- 为什么83%准确率仍可能没有用：如果数据中83%的用户都不流失，那模型只要永远预测"不流失"就能达到83%准确率，但它完全没有识别出任何流失风险，对业务毫无价值
